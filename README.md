# doodle-recog
Doodle's are fun, and so is detecting them.

# Steps to run
```
Step 1: git pull origin file
Step 2: open terminal in vs code
Step 2: pip install -r requirements.txt
Step 3: python app.py (or python3 app.py [whichever works])
```
